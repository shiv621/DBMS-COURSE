\section{ABSTRACT}

Many scholars are studying the problem of processing XML documents and optimising XML queries as the need for manipulating and exchanging XML data grows.
However, in the context of XML database systems, there are still numerous outstanding difficulties in query optimization. In this study, we analyse the numerous optimization strategies developed by prior research and used by existing XML database systems to present an overview of the state of XML query optimization.
